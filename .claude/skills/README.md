# Skills

Your starter skills live here. They work whenever Claude Code is pointed at this vault.

- vault-setup              personalize the Script by interview (run this first)
- advanced-script-interview deepen the Script after Obsidian and the link map work
- see-your-setup            audit your Script, your skills, and what is loaded
- security-check           check installs, backups, sharing, permissions, and risky moves
- dashboard-setup           a dashboard of what is compounding across sessions
- plan-as-page             render any plan as a clean, shareable page
- convince-me              make Claude restate intent before acting, and explain after
- adversarial-review       a find-whats-wrong pass before you ship
- reflect                  end-of-session retrospective that deposits lessons
- remember-this            promote a rule or fact into the right file mid-session
- handoff                  save restart context before sleep, compaction, or session switch
- save-as-skill            turn a move you have done twice into a reusable skill
- create-workflow-diagram  turn a plain-text process into a shareable diagram
