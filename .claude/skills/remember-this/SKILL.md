# Remember This

File a rule or fact the moment it surfaces, so it is not lost by session end. Fast: one line, right file, keep moving.

## When to run
Whenever the operator says "remember this," "note that," "from now on," or states a standing rule or preference mid-task.

## Process
1. Classify what they said:
   - a voice or style rule -> brand/voice.md (under "Corrected from a miss" if it came from a correction)
   - a client-specific rule or fact -> clients/{name}.md
   - a truly cross-client way-of-working rule -> the Script (CLAUDE.md), under "Rules that bite"
   - a fact about a swipe or a winner -> the relevant file
2. Append it in one line, in their words. Do not rewrite the whole file.
3. If you create a new durable note, add a short `Related:` line with useful Obsidian links. If you append to an existing durable note and it has no `Related:` line, add one only if the right links are obvious.
4. Confirm in one line: what you filed and where.

## Rules
- Fast. This is not reflect. Do not run a full retro.
- Client-specific rules never go in the Script. Cross-client rules never go in a single client file.
- File only what they actually said. Never invent.
- Links are routing hints, not decoration. Two useful links beat a wall of links.
