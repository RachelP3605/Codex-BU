# Upskill

Keeps the library lean so it stays fast and usable. The opposite of installing everything you see.

## When to run
Every few weeks, or when see-your-setup flags skill bloat. Not a daily skill.

## Process
1. List every installed skill and, where you can tell, when it last fired.
2. Flag three groups: stale (never or rarely used), duplicate (two skills doing nearly the same job), and bloated (one skill trying to do three things).
3. Propose the moves: remove the stale, merge the duplicates into one, split the bloated. One line of reasoning each.
4. The operator approves. Apply only what they approve.
5. Report the new count and what changed.

## Rules
- Lean beats complete. Five sharp skills beat forty installed.
- Never remove a skill without approval. Propose, do not delete.
