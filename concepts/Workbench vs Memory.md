# Workbench vs Memory

Keep the work and the lesson separate.

## Workbench

The workbench is where finished work lives:

- `copy/`
- `creatives/`

## Memory

Memory is where the reusable lesson lives:

- `winners/`
- `losers/`
- `brand/`
- `clients/`
- `workflows/`

Example: save the ad in `creatives/`, then promote why it worked into `winners/`.
