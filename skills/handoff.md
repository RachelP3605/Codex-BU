# handoff

Command: `/handoff`

Use this before switching sessions, before compaction, when the thread is getting too long, or when you want to restart tomorrow without re-explaining everything.

It writes the current state, decisions, changed files, open loops, exact next step, and a resume prompt into today's daily note or a project handoff file.

Use it for continuity, not memory promotion.

Actual runtime file: `.claude/skills/handoff/SKILL.md`
