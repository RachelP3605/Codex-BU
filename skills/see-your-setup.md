# see-your-setup

Command: `/see-your-setup`

Use this when you want to check the workspace.

It reports the current Script, lists the installed skills, checks the Obsidian link layer, and shows what folders are filling up.

This is the quickest way to answer: "What did I install, and is it still clean?"

Actual runtime file: `.claude/skills/see-your-setup/SKILL.md`
